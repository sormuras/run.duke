# Duke

*Java's mascot and mechanic*

![duke-tools.png](doc/duke-tools.png)

## Building from Source

- Install [JDK] 17 or later.
- Compile and test `run.duke` module:
  ```shell
  java @bootstrap
  ```
- Run `versions` tool:
  ```shell
  java @duke versions
  ```


[JDK]: https://www.oracle.com/java/technologies/downloads/
